# Mailcow WHMCS Module

Modern WHMCS server module for provisioning and managing mailboxes in [mailcow](https://mailcow.email/) via its API.

## Overview

This project provides a simple and reliable integration between WHMCS and mailcow, allowing automated mailbox provisioning, suspension, termination, and management directly from WHMCS.

## Status

🚧 Work in progress  
This is a fresh, cleaned-up continuation of earlier community efforts.
